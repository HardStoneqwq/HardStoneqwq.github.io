51单片机入门

发表于 2021-08-27 更新于 2021-08-30 分类于 hardware

51单片机是对兼容Intel8051指令系统的单片机的统称。51单片机广泛应用于家用电器、汽车、工业测控、通信设备中。通过 51单片机入门教程-bilibili 进行51单片机的入门学习。

准备

  1. 硬件：51单片机开发板（普中）

  2. 软件

    • Keil5:C51

      • Project->New uVision Project->新建项目路径(~\Keil\2-1 LED)->新建工程名(project)->选择器件(Microchip->AT89C52)->Source Group 1->Add New Item to Group “Source Group 1”->main.c

      • Build

      • Options for Target -> Output -> Create HEX File

    • STC-ISP

      安装驱动:USB连接开发板->在stc安装目录下 USB to UART Driver->CH340_CH341->ch341ser.exe->Install

      普中开发板:MUC Type(STC89C52RC/LE52RC)->Open Code File->Download -> 重新启动MCU

单片机概览

  1. 单片机（Single-Chip Microcomputer/Micro Controller Unit/MCU）集成了CPU（中央处理器）、RAM（随机存储器）、ROM（只读存储器）、I/O口、中断系统、定时器等功能。
  2. 单片机功能：信息采集（依靠传感器）、处理（依靠CPU）和硬件设备（如电机、LED等）的控制。
  3. 应用领域:智能仪表、实时工控、通讯设备、导航系统、家用电器等。

初识51单片机

  1. 主要品牌

    • Intel（英特尔）：80C31、80C51、87C51，80C32、80C52、87C52等；

    • Atmel（艾特梅尔）：89C51、89C52、89C2051，89S51(RC)，89S52(RC)等；

    • STC（宏晶）：STC89C51、STC90C51、STC11系列、STC15系列、STC8系列等。

  2. STC89C52单片机

    • 位数:8位
    • RAM:512字节
    • ROM:8K(Flash)
    • 工作频率:12MHZ
